This directory contains the Phone Directory application implementation
using a map. Two versions are included: the alt version uses the map index
operator and the non-alt version uses map::insert.

Makefile		Makefile for g++ 

makefile.mak		Makefile for Visual Studio .NET

Map_Based_PD.cpp	Implemention of the Phone Directory using a map

Map_Based_PD.h		Definition of the Phone Directory using a map

PD_Application.cpp	The the Phone Directory application main program

phone.dat		The phone directory data
